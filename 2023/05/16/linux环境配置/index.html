linux环境下配置环境-java

jdk环境配置

  • 下载jdk的安装包上传到指定目录

  • 解压安装包

    • tar -zxvf jdk-8u261-linux-x64.tar.gz

  • 检查是否配置成功

    • java -version

  • 往配置文件里面添加环境变量

    • vim /etc/profile  编辑配置文件

MAVEN_HOME=/usr/local/maven/apache-maven-3.5.4
PATH=$MAVEN_HOME/bin:$PATH
export MAVEN_HOME PATH

重新加载配置文件
source /etc/profile

git环境安装

  • 官网下载安装包

    • https://git-scm.com/download/linux

    • 卸载掉插件自带的低版本git

    • 编译源文件

    • 安装

  • 编辑环境变量 

    • vim /etc/profile

    • 重新加载配置文件

    • source /etc/profile

nginx安装部署

  • 下载linux安装包 上传到指定目录

  • 解压安装包

    • tar -xzvf nginx-1.16.1.tar.gz

  • 编译文件 

    • make

  • 需要注意的是编译后的文件在 nginx-1.16.1.tar.gz同级目录下

  • 以配置文件的方式启动

    • ./nginx -c /usr/local/nginx/conf/nginx.conf
